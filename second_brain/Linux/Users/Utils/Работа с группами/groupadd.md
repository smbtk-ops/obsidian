### ** Основные ключи команды groupadd**

|**Ключ**|**Название**|**Описание**|**Пример использования**|
|---|---|---|---|
|-f|force|Не выдает ошибку, если группа уже существует|groupadd -f devs|
|-g|GID|Устанавливает числовой идентификатор группы (GID)|groupadd -g 1001 devs|
|-K|параметр конфигурации|Устанавливает временное значение параметра из /etc/login.defs|groupadd -K GID_MIN=2000 newgroup|
|-o|allow duplicate GID|Разрешает дублирование GID (только с -g)|groupadd -o -g 1001 sharedgroup|
|-p|зашифрованный пароль|Устанавливает зашифрованный пароль группы _(не рекомендуется)_|groupadd -p '$6$salt$hash...' devs|
|-r|system group|Создает системную группу (обычно с GID < 1000)|groupadd -r syslog|

### **Примеры:**

1. **Создать обычную группу admins:**
```
groupadd admins
```
    
1. **Создать группу с конкретным GID:**
```
groupadd -g 1050 projectX
```
    
2. **Создать системную группу:**
```
groupadd -r backup
```
    
3. **Создать группу, даже если она уже существует (без ошибки):**
```
groupadd -f admins
```

4. **Создать группу с дублирующимся GID:**
```
groupadd -o -g 1001 duplicategroup
```
